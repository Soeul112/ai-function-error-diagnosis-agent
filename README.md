# AI 错题诊断 Prompt 评测小项目

这个项目用于对齐 AI 产品经理实习 JD 中的“模型选型、业务效果评测、自动化评测”要求。场景选择高中函数错题诊断：给定题目、学生答案和标准答案，让大模型输出错因识别、讲解反馈和下一步学习建议。

## 交付物

- `data/prompt_eval_cases.csv`：25 条测试样本，覆盖典型错题、边界题和异常输入。
- `prompts/prompt_versions.md`：2 个 Prompt 版本，分别为基础版 V1 和结构化版 V2。
- `scripts/evaluate_prompt_results.py`：自动汇总脚本，统计错因准确率、相关性、结构化完整率和幻觉风险。
- `outputs/ai_math_prompt_eval.xlsx`：Excel 对比表，包含总览、评测数据和 Prompt 版本。

## 样本设计

测试集共 25 条：

- 典型错题 16 条：定义域遗漏、单调性误判、零点遗漏、图像平移方向错误、复合函数理解错误等。
- 边界题 4 条：答案正确但过程不完整、参数题、限定区间正确、口语化但含义正确。
- 异常输入 5 条：空答案、乱码、无关输入、题干信息不足等。

## 评测指标

1. 错因识别准确性：是否命中预期错因，0/1。
2. 回复相关性：是否围绕题目和学生错误展开，1-5 分。
3. 结构化输出完整率：是否包含必填字段，0-1。
4. 幻觉风险：是否编造题干不存在的信息或错误知识点，0/1。

## 如何使用

1. 用 Prompt V1 和 V2 分别跑 25 条样本。
2. 将模型输出填入 Excel 的 `Eval_Data` 表。
3. 按四个指标进行人工或半自动评分。
4. 运行脚本查看汇总：

```bash
python scripts/evaluate_prompt_results.py
```

## 简历表述

面向高中函数错题诊断场景，搭建 25 条 Prompt 评测集，覆盖典型错题、边界题和异常输入；设计基础版与结构化版 2 套 Prompt，对模型输出从错因识别准确性、回复相关性、结构化完整率和幻觉风险 4 个维度进行人工/半自动评测，并用 Excel/Python 汇总对比结果，基于评测结论迭代提示词策略，提升错因定位稳定性与诊断报告可用性。
