# 简历项目描述备用

**高一函数 AI 错题诊断与家长反馈提效工具 | AI 教育产品个人项目**  
2026.06

- 面向高一数学家教场景，设计并实现一个本地可运行的 AI 教育产品原型，覆盖错题诊断、AI 家教式追问、变式题二次作答、家长反馈生成和老师端汇总。
- 使用 React + Vite 搭建多页面前端，使用 Node.js + Express 提供本地 API，并基于 Node 24 内置 `node:sqlite` 保存学生、题库、原题作答、诊断报告和变式题作答记录。
- 自建 20 道中文高一函数题库，覆盖定义域、值域、单调性、奇偶性、零点 5 个模块；每道题包含知识点、标准答案、常见错误、错因标签、追问话术和变式题模板。
- 设计本地规则引擎模拟 AI 教学 Agent 能力，根据学生答案生成追问、提示、错因归类、薄弱知识点、补救路径和家长可读反馈。
- 新增“变式迁移”闭环：学生生成变式题后必须二次作答，系统记录正确率并判断“迁移掌握/仍需巩固”，避免项目停留在只讲原题的 Demo。
- 设计老师端一键复制家长反馈能力，将本次错因、薄弱点、下一步计划和监督建议自动整合为可发送文案，回应家教老师课后反馈耗时的真实痛点。
- 调研 MathDial、Khan tutoring accuracy dataset、CMM-Math 等开源资料后，采用自建小样本 + 可扩展数据结构的方式适配中文高一函数错因诊断场景。

## 简历精简版

高一函数 AI 错题诊断与家长反馈提效工具：基于 React + Node.js + SQLite 实现本地 AI 教育产品原型，自建 20 道高一函数题库和错因标签体系，跑通“学生错题作答 - AI 追问诊断 - 变式题二次作答 - 家长反馈生成 - 老师端汇总”闭环；通过规则引擎模拟 AI 家教能力，并沉淀原题/变式作答数据，用于判断学生是否真正完成同类题迁移。
